获取导航信息

请求地址: http://yapi.demo.qunar.com/mock/5280/nav

请求方法: GET

请求参数:

参数名 类型 是否必须 示例 描述
size int 否 -1 获取导航标题的个数，-1表示获取全部，默认获取全部

获取文章列表

请求地址: http://yapi.demo.qunar.com/mock/5280/list

请求方法: POST

请求参数:

参数名 类型 是否必须 示例 描述
page int 是 0 获取某一页的文章
size int 否 10 获取一页文章的文章数，默认返回10条文章
typeId int 是 3 文章类型
